## 7. 开发阶段详细实施方案

### 7.1 第一周（第 7 周）：用户管理模块

**目标**: 完成用户 CRUD + 持久化 + UI

```
Day 1-2: UserManager.js
  - electron-store 初始化
  - getUsers / saveUser / deleteUser
  - UUID 生成
  
Day 3: userHandlers.js (IPC)
  - 注册 IPC handler
  - 错误处理
  
Day 4-5: UserManager.jsx (UI)
  - 用户列表展示
  - 添加/编辑表单 (Ant Design Form)
  - 删除确认对话框
  - 单元测试
```

### 7.2 第二周（第 8 周）：会话管理 + 浏览器基础

```
Day 1-2: SessionManager.js
  - session.fromPartition 创建隔离会话
  - 会话生命周期管理
  - 会话销毁前存在性检查
  
Day 3: 浏览器窗口基础
  - BrowserWindow 创建
  - webview 标签配置
  - partition 绑定用户会话
  
Day 4-5: 标签页管理
  - TabBar 组件
  - 标签页创建/关闭/切换
  - webview 事件监听
```

### 7.3 后续周次（类似模式）

每周按相同模式推进：**模块逻辑 → IPC → UI → 测试**

## 9. 立即可执行的下一步

1. **创建 GitHub 仓库** `mmmmh2o/multi-user-browser`
2. **初始化项目骨架**:
   ```bash
   npm init -y
   npm install electron electron-store electron-log electron-updater
   npm install react react-dom react-router-dom @reduxjs/toolkit react-redux antd
   npm install -D vite @vitejs/plugin-react electron-builder
   npm install -D jest @testing-library/react eslint prettier
   ```
3. **配置 electron-builder.yml**
4. **编写 main.js 入口 + preload.js**
5. **启动第一个模块：UserManager**
